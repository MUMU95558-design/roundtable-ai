### view-roundtable（圆桌讨论 — 主功能）

`view-roundtable` 为 **fixed 全屏覆盖层**，叠加在其他视图之上，包含左侧固定侧边栏 + 右侧消息区。

```
┌── 左侧边栏（fixed，~240px）────┬──── 右侧消息流（可滚动）────────────────────┐
│                               │                                             │
│  [+ 开启新构思]               │              ┌──────────────────────────┐   │
│  ─────────────────────       │              │ 你能做什么？          21:36│   │  ← 用户气泡
│  [🏛 场景会议室]              │              └──────────────────────────┘   │
│  [★ 收藏夹]                  │                                             │
│  [👤 个人中心]               │  ┌── 查理·芒格 ── [● ● ● Streaming] ─────┐  │
│  ─────────────────────       │  │ 先反过来想：如何确保效率低下？...▍      │  │  ← streaming
│  历史记录                     │  └─────────────────────────────────────────┘  │
│  ┌──────────────────────┐   │                                             │
│  │ 如何提高工作效率      │   │  ┌── 阿尔弗雷德·阿德勒 ── [连接中…] ──────┐  │
│  │ 5天前               │   │  │  （骨架占位）                            │  │  ← waiting
│  └──────────────────────┘   │  └─────────────────────────────────────────┘  │
│  ┌──────────────────────┐   │                                             │
│  │ 你能做什么           │   │                                             │
│  │ 今天                │   ├─────────────────────────────────────────────────┤
│  └──────────────────────┘   │  ┌─────────────────────────────────────────┐   │
│  ─────────────────────       │  │ 🌐 ⚔ 输入问题 · Enter 发送 · @单独提问 [↑]│   │  ← 浮动输入框
│  [⚙ 设置]                   │  └─────────────────────────────────────────┘   │
│  [← 返回首页]               │                                             │
└─────────────────────────────┴─────────────────────────────────────────────┘
```

输入框工具栏图标说明：
- `🌐` — 联网搜索开关
- `⚔` — 辩论模式开关

**Agent 消息卡片状态：**

| 状态 | 视觉 | Badge |
|------|------|-------|
| `waiting` | 完整色，有头像/名字/框，骨架占位 | `连接中…`（Agent 专属色） |
| `streaming` | 玻璃高亮 + 彩色阴影 + 打字光标 ▍ | `● ● ● Streaming`（三点 bounce） |
| `completed` | 正常内容 | 无 badge |

**@mention 下拉（输入 @ 触发）：**
```
┌──────────────────────┐
│ 单独提问              │  ← JetBrains Mono 10px
├──────────────────────┤
│ 查理·芒格   ← 蓝色高亮 │
│ 阿尔弗雷德·阿德勒     │
│ 安德烈·卡帕西         │
│ 埃隆·马斯克           │
├──────────────────────┤
│ ↑↓ 导航 · Enter 选中  │
└──────────────────────┘
  定位：absolute bottom-full，出现在输入框上方
```

### 玻璃功能卡片（glass-card）

```
┌─────────────────────────────────┐
│  [图标]                         │
│  标题（20px 700）                │
│  描述文字（14px muted）          │
└─────────────────────────────────┘
  border-radius: 32px
  cursor: pointer（可点击的卡片有 hover/active 效果）
  hover: translateY(-2px) + shadow 加深
  active: scale(0.98)
```

## 页面间距规范

| 元素 | 值 |
|------|-----|
| 导航栏高度 | ~64px |
| 主内容区顶部间距 | `pt-32`（128px）|
| 历史/镜像页面顶部 | `margin-top: -27px`（相对 pt-32 缩短约 30%）|
| 内容区左右 padding | `32px` |
| 卡片间距 | `16px`（历史列表）/ `32px`（功能区块）|

## 已知限制（待迭代）

| 问题 | 说明 |
|------|------|
| 字体/图标依赖 Google CDN | 网络差时加载慢，可本地化 |
| Tailwind CDN 运行时编译 | 初次加载轻微延迟 |
| 认知镜像话题 tag | 显示完整标题而非 NLP 提取关键词 |
| 无移动端适配 | 桌面优先，< 768px 未优化 |
